# 油气管道监测管理系统

## 项目概述

油气管道监测管理系统是一个综合性的工业物联网监测平台，专门用于油气管道的实时监控、数据分析、任务管理和应急响应。系统采用前后端分离架构，支持Web端管理和移动端操作，为管道运维提供全方位的数字化解决方案。

## 系统架构

```
pipeline_management_system/
├── backstage-vue/          # Vue.js Web端管理系统 (模块化架构)
│   ├── src/
│   │   ├── modules/        # 功能模块
│   │   │   ├── auth/       # 认证模块
│   │   │   ├── dashboard/  # 仪表板模块
│   │   │   ├── monitoring/ # 监控模块
│   │   │   ├── management/ # 管理模块
│   │   │   ├── emergency/  # 应急模块
│   │   │   ├── logs/       # 日志模块
│   │   │   └── shared/     # 共享组件和工具
│   │   ├── router/         # 路由配置
│   │   ├── store/          # 全局状态管理
│   │   └── main.js         # 应用入口
│   └── vue.config.js       # Vue配置 (包含模块路径别名)
├── repairman-uniapp/       # UniApp移动端应用
├── server/                 # Java Spring Boot后端服务
├── utils/                  # 工具脚本和数据处理
└── 相关文件/               # 项目文档和设计资料
```

## 核心功能模块

### 🎯 数据可视化模块

- **实时监控大屏**: 基于ECharts的数据可视化展示

- **任务详情**: 检修任务状态和进度跟踪

### 📊 数据管理模块

- **监测数据管理**: 传感器数据的存储、查询和分析

- **任务管理**: 检修任务的创建、分配和跟踪
- **检修员信息**: 检修人员的基本信息和绩效管理

### 🤖 虚拟专家系统

- **智能分析**: 基于WebSocket的实时数据分析

### 🎮 模拟与演练

- **事故模拟**: 各种管道事故场景的模拟演练
- **应急演练**: 应急响应流程的训练和评估

### 🚨 事故响应

- **实时报警**: 异常数据的自动报警机制

- **操作日志**: 系统操作的完整记录

- **系统监控**: 系统运行状态和性能监控

## 技术栈详解

### 数据库设计

#### 核心数据表

- **area**: 区域表 (3209条记录) - 省市区三级行政区划
- **pipe**: 管道表 (2000条记录) - 管道基础信息
- **sensor**: 传感器表 (30000条记录) - 传感器设备信息
- **inspection**: 监测数据表 (1000000条记录) - 传感器监测数据
- **repairman**: 检修员表 (500条记录) - 检修人员信息
- **task**: 任务表 (50000条记录) - 检修任务信息

#### 用户认证表

- **admin_registration**: 管理员登录表 (10条记录)
- **repairman_registration**: 检修员登录表 (500条记录)

#### 业务扩展表

- **manoeuvre**: 演习表 (100条记录) - 演练活动记录
- **conn_manoeuvre_repairman**: 演习检修员关联表 (500条记录)
- **log**: 日志表 (100000条记录) - 系统操作日志

## 快速开始

### 环境要求

- **Java**: JDK 17+
- **Node.js**: 14.0+
- **MySQL**: 8.0+
- **Python**: 3.8+ (工具脚本)

### 安装部署

#### 1. 数据库初始化

```sql
-- 创建数据库
CREATE DATABASE pipeline_management_system;

-- 导入数据表结构
source 相关文件/数据库表/pipeline_monitoring_system.sql;

-- 导入区域数据
source 相关文件/数据库表/insert_area.sql;
```

#### 2. 后端服务启动

```bash
cd server

# Web端服务
cd server-web
mvn spring-boot:run

# 移动端服务
cd server-mobile
mvn spring-boot:run
```

#### 3. 前端应用启动

```bash
# Web端管理系统
cd backstage-vue
npm install
npm run serve

# 移动端应用
cd repairman-uniapp
npm install
npm run dev:h5
```

#### 4. 数据生成 (可选)

```bash
cd utils/generateData
pip install -r requirements.txt
python main.py all
```
