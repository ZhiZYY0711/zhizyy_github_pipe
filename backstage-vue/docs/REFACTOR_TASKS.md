## 📄 详细设计

### 1. 数据大屏（Dashboard）

**优先级**: ⭐⭐⭐ 高

**页面路径**: `/dashboard`

**设计理念**: 全屏数据墙，实时监控，一目了然

#### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  页面标题 + 时间 + 刷新按钮                           │
├─────────────────────────────────────────────────────┤
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐      │
│  │传感器│ │异常数│ │警告数│ │进行中│ │超时数│      │
│  │ 150  │ │  5   │ │  12  │ │  8   │ │  2   │      │
│  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘      │
├─────────────────────────────────────────────────────┤
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐│
│  │              │ │              │ │              ││
│  │ 任务状态分布 │ │ 任务数量趋势 │ │ 区域任务对比 ││
│  │   (饼图)     │ │   (折线图)   │ │   (柱状图)   ││
│  │              │ │              │ │              ││
│  └──────────────┘ └──────────────┘ └──────────────┘│
├─────────────────────────────────────────────────────┤
│  ┌────────────────────┐ ┌────────────────────────┐ │
│  │                    │ │                        │ │
│  │    KPI排行榜       │ │    流水报警列表        │ │
│  │  (检修员排名)      │ │  (实时报警滚动)        │ │
│  │                    │ │                        │ │
│  └────────────────────┘ └────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

#### 核心组件

1. **顶部指标卡区域**
   - 5个 MetricCard 组件
   - 显示关键KPI数据
   - 支持点击钻取

2. **图表区域**
   - 饼图：任务状态分布（ECharts）
   - 折线图：任务数量趋势（ECharts）
   - 柱状图：区域任务对比（ECharts）

3. **排行榜区域**
   - KPI排行榜（表格形式）
   - 流水报警列表（滚动列表）

#### 数据接口

- `GET /data_visualization/all/whole_kpi` - 全局KPI
- `POST /data_visualization/task_details/task_status` - 任务状态分布
- `POST /data_visualization/task_details/task_nums` - 任务数量趋势
- `POST /data_visualization/task_details/task_contrast` - 区域任务对比
- `POST /data_visualization/task_details/KPI_list` - KPI排行榜
- `GET /data_visualization/all/running_water_alarm` - 流水报警

#### 交互设计

1. **自动刷新**: 每30秒自动刷新数据
2. **点击钻取**: 点击指标卡跳转到详情页
3. **图表交互**: 支持图表缩放、筛选
4. **报警提示**: 新报警闪烁提示

### 标签页工作区逻辑

#### 标签页管理

```javascript
// 标签页数据结构
tabs: [
  { path: '/dashboard', title: '数据大屏', closable: false },
  { path: '/tasks', title: '任务管理', closable: true },
  { path: '/equipment', title: '设备管理', closable: true }
]

// 添加标签页
addTab(route) {
  if (!route.meta || !route.meta.title) return;
  
  const exists = this.tabs.find(tab => tab.path === route.path);
  if (!exists) {
    this.tabs.push({
      path: route.path,
      title: route.meta.title,
      closable: route.path !== '/dashboard' // 首页不可关闭
    });
  }
}

// 关闭标签页
closeTab(tab) {
  const index = this.tabs.findIndex(t => t.path === tab.path);
  this.tabs.splice(index, 1);
  
  // 如果关闭的是当前页面，跳转到相邻标签页
  if (this.$route.path === tab.path && this.tabs.length > 0) {
    const nextTab = this.tabs[index] || this.tabs[index - 1];
    this.$router.push(nextTab.path);
  }
}
```

### 路由守卫逻辑

#### 全局前置守卫

```javascript
// router/index.js
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token');
  
  // 白名单路由
  const whiteList = ['/login'];
  
  if (token) {
    // 已登录，访问登录页则跳转到首页
    if (to.path === '/login') {
      next('/dashboard');
    } else {
      next();
    }
  } else {
    // 未登录
    if (whiteList.includes(to.path)) {
      next();
    } else {
      next('/login');
    }
  }
});
```
