# 数据可视化模块 API

## 概述

数据可视化模块提供全局KPI指标、任务详情分析、数据监控等功能，用于数据大屏展示。

---

## 1. 全局KPI指标

### 1.1 获取全局KPI数据

#### 接口说明

获取系统全局关键性能指标数据，包括传感器数量、警告数量、任务状态等。

#### 请求信息

- **URL**: `/data_visualization/all/whole_kpi`
- **Method**: `GET`
- **认证**: 需要

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| area_id | Long | 否 | 区域ID，不传则查询全局 |

#### 响应参数

```json
{
  "code": 200,
  "message": "获取全局KPI数据成功",
  "data": {
    "sensor_numbers": 150,
    "abnormal_sensor_numbers": 5,
    "warnings": 12,
    "underway_task": 8,
    "overtime_task": 2
  }
}
```

| 参数名 | 类型 | 说明 |
|--------|------|------|
| sensor_numbers | Long | 传感器总数 |
| abnormal_sensor_numbers | Long | 异常传感器数量 |
| warnings | Long | 警告数量 |
| underway_task | Long | 进行中的任务数量 |
| overtime_task | Long | 超时任务数量 |

## 2. 任务详情分析

### 2.1 获取任务状态分布

#### 接口说明

获取不同状态的任务数量分布，用于饼图展示。

#### 请求信息

- **URL**: `/data_visualization/task_details/task_status`
- **Method**: `POST`
- **认证**: 需要

#### 请求参数

```json
{
  "area_id": 130100,
  "start_time": 1704067200000,
  "end_time": 1704153600000
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| area_id | Long | 是 | 区域ID（6位数字） |
| start_time | Long | 否 | 开始时间戳（毫秒） |
| end_time | Long | 否 | 结束时间戳（毫秒） |

#### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "status": "待处理",
      "count": 15
    },
    {
      "status": "进行中",
      "count": 8
    },
    {
      "status": "已完成",
      "count": 42
    },
    {
      "status": "已取消",
      "count": 3
    }
  ]
}
```

## 前端集成建议

### 1. 数据大屏布局

```vue
<template>
  <div class="dashboard">
    <!-- 顶部指标卡 -->
    <div class="kpi-cards">
      <KPICard title="传感器总数" :value="kpiData.sensor_numbers" />
      <KPICard title="异常传感器" :value="kpiData.abnormal_sensor_numbers" type="warning" />
      <KPICard title="警告数量" :value="kpiData.warnings" type="danger" />
      <KPICard title="进行中任务" :value="kpiData.underway_task" />
    </div>
    
    <!-- 图表区域 -->
    <div class="charts-row">
      <PieChart title="任务状态分布" :data="taskStatusData" />
      <LineChart title="任务数量趋势" :data="taskTrendData" />
      <BarChart title="区域任务对比" :data="areaTaskData" />
    </div>
  </div>
</template>
```

### 2. 实时数据更新

```javascript
// 使用定时器定期刷新数据
mounted() {
  this.fetchKPIData();
  this.timer = setInterval(() => {
    this.fetchKPIData();
  }, 30000); // 每30秒刷新一次
},
beforeDestroy() {
  clearInterval(this.timer);
}
```
