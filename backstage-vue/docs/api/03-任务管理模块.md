# 任务管理模块 API

## 概述

任务管理模块提供任务的增删改查功能，支持分页查询、条件筛选等。

---

## 1. 分页查询任务

### 接口说明

分页查询任务列表，支持多条件筛选。

### 请求信息

- **URL**: `/data_management/tasks/find_task_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "page": 1,
  "page_size": 10,
  "inspection_id": 1,
  "repairman_id": 1,
  "area_id": 130100,
  "pipe_id": 1,
  "type": 1,
  "priority": 1,
  "status": 1,
  "start_time": "2024-01-01 00:00:00",
  "end_time": "2024-01-31 23:59:59"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | Integer | 否 | 页码，从1开始 |
| page_size | Integer | 否 | 每页数量，默认10 |
| inspection_id | Long | 否 | 检查ID |
| repairman_id | Long | 否 | 检修员ID |
| area_id | Long | 否 | 区域ID |
| pipe_id | Long | 否 | 管道ID |
| type | Integer | 否 | 任务类型 |
| priority | Integer | 否 | 优先级 |
| status | Integer | 否 | 任务状态 |
| start_time | String | 否 | 开始时间 |
| end_time | String | 否 | 结束时间 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": "1",
        "title": "管道A区压力异常检测",
        "type": "检修",
        "area_name": "石家庄市",
        "pipe_name": "主管道A",
        "priority": "紧急",
        "status": "进行中",
        "assignee": "张三",
        "phone": "13800138000",
        "public_time": "2024-01-15 10:30:00",
        "response_time": "2024-01-15 10:35:00",
        "accomplish_time": null,
        "feedback_information": "正在检修中"
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```

| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | String | 任务唯一标识 |
| title | String | 任务标题 |
| type | String | 任务类型 |
| area_name | String | 任务区域 |
| pipe_name | String | 任务所属管道 |
| priority | String | 优先级（紧急/高/中/低） |
| status | String | 任务状态（待处理/进行中/已完成/已取消） |
| assignee | String | 指派检修员姓名 |
| phone | String | 联系方式 |
| public_time | String | 发布时间 |
| response_time | String | 响应时间 |
| accomplish_time | String | 任务完成时间 |
| feedback_information | String | 检修员反馈信息 |

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getTasks(params) {
  return request({
    url: '/data_management/tasks/find_task_params',
    method: 'post',
    data: params
  });
}

export function getTaskById(id) {
  return request({
    url: '/data_management/tasks/find_task_id',
    method: 'post',
    data: { id }
  });
}

export function createTask(data) {
  return request({
    url: '/data_management/tasks/add_task',
    method: 'post',
    data
  });
}

export function updateTask(data) {
  return request({
    url: '/data_management/tasks/update_task',
    method: 'post',
    data
  });
}

export function deleteTask(id) {
  return request({
    url: `/data_management/tasks/delete_task/${id}`,
    method: 'get'
  });
}

export function getTaskIndicators(areaId) {
  return request({
    url: '/data_management/tasks/Indicator_card',
    method: 'get',
    params: { area_id: areaId }
  });
}
```
