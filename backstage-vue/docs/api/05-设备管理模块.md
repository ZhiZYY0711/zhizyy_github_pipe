# 设备管理模块 API

## 概述

设备管理模块提供设备信息的增删改查功能。

---

## 1. 查询设备（非ID）

### 接口说明

查询设备列表，支持多条件筛选。

### 请求信息

- **URL**: `/data_management/equipment/find_equipment_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "id": "",
  "type": "传感器",
  "status": "正常",
  "area_id": "130100",
  "pipe_id": "1",
  "responsible": "张三",
  "last_check_start": "2024-01-01",
  "last_check_end": "2024-12-31",
  "page": "1",
  "page_size": "10"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 否 | 设备编号 |
| type | String | 否 | 设备类型 |
| status | String | 否 | 设备状态（正常/故障/维护中/离线） |
| area_id | String | 否 | 设备所属区域ID |
| pipe_id | String | 否 | 设备所属管道ID |
| responsible | String | 否 | 负责人 |
| last_check_start | String | 否 | 最后检查起始时间 |
| last_check_end | String | 否 | 最后检查结束时间 |
| page | String | 否 | 页码 |
| page_size | String | 否 | 每页数据量 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": "1",
      "type": "压力传感器",
      "model": "PT-100",
      "status": "正常",
      "area_name": "石家庄市",
      "pipe_name": "主管道A",
      "responsible": "张三",
      "install_date": "2023-01-15",
      "last_check": "2024-01-10"
    }
  ]
}
```

```vue
      <el-table-column prop="model" label="型号" />
      <el-table-column prop="status" label="状态">
        <template slot-scope="scope">
          <status-tag :text="scope.row.status" :type="getStatusType(scope.row.status)" />
        </template>
      </el-table-column>
      <el-table-column prop="area_name" label="所属区域" />
      <el-table-column prop="responsible" label="负责人" />
      <el-table-column prop="last_check" label="最后检查时间" />
    </el-table>

<script>
export default {
  methods: {
    getStatusType(status) {
      const map = {
        '正常': 'success',
        '故障': 'danger',
        '维护中': 'warning',
        '离线': 'default'
      };
      return map[status] || 'default';
    }
  }
};
</script>
```

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getEquipment(params) {
  return request({
    url: '/data_management/equipment/find_equipment_params',
    method: 'post',
    data: params
  });
}

export function getEquipmentById(id) {
  return request({
    url: '/data_management/equipment/find_equipment_id',
    method: 'post',
    data: { id }
  });
}

export function addEquipment(data) {
  return request({
    url: '/data_management/equipment/add_equipment',
    method: 'post',
    data
  });
}

export function updateEquipment(data) {
  return request({
    url: '/data_management/equipment/update_equipment',
    method: 'post',
    data
  });
}

export function deleteEquipment(ids) {
  return request({
    url: `/data_management/equipment/delete_equipment/${ids}`,
    method: 'get'
  });
}

export function getEquipmentIndicators(areaId) {
  return request({
    url: '/data_management/equipment/Indicator_card',
    method: 'get',
    params: { area_id: areaId }
  });
}
```
