# 监测数据模块 API

## 概述

监测数据模块提供管道监测数据的增删改查功能，包括压力、流量、温度、震动等数据。

---

## 1. 查询监测数据（非ID）

### 接口说明

分页查询监测数据，支持多条件筛选。

### 请求信息

- **URL**: `/data_management/monitoring_data/find_data_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "min_pressure": "0.5",
  "max_pressure": "1.5",
  "min_flow": "100",
  "max_flow": "200",
  "min_temperature": "20",
  "max_temperature": "30",
  "min_vibration": "0",
  "max_vibration": "0.1",
  "data_status": "0",
  "sensor_id": "1",
  "pipeline_name": "主管道A",
  "monitor_start_time": "2024-01-01 00:00:00",
  "monitor_end_time": "2024-01-31 23:59:59",
  "page": 1,
  "page_size": 10
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| min_pressure | String | 否 | 最小压力 |
| max_pressure | String | 否 | 最大压力 |
| min_flow | String | 否 | 最小流量 |
| max_flow | String | 否 | 最大流量 |
| min_temperature | String | 否 | 最小温度 |
| max_temperature | String | 否 | 最大温度 |
| min_vibration | String | 否 | 最小震动 |
| max_vibration | String | 否 | 最大震动 |
| data_status | String | 否 | 数据状态（0安全 1良好 2危险 3高危） |
| sensor_id | String | 否 | 传感器ID |
| pipeline_name | String | 否 | 管道名称 |
| monitor_start_time | String | 否 | 监测起始时间 |
| monitor_end_time | String | 否 | 监测结束时间 |
| page | Integer | 否 | 页码 |
| page_size | Integer | 否 | 每页大小 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": "1",
        "sensor_id": "S001",
        "sensor_name": "压力传感器A",
        "pipeline_name": "主管道A",
        "pressure": 0.85,
        "flow": 120.5,
        "temperature": 25.3,
        "vibration": 0.02,
        "data_status": "安全",
        "monitor_time": "2024-01-15 10:30:00"
      }
    ],
    "total": 1000,
    "page": 1,
    "pageSize": 10
  }
}
```

## 5. 删除监测数据

### 接口说明

批量删除监测数据记录。

### 请求信息

- **URL**: `/data_management/monitoring_data/delete_data/{ids}`
- **Method**: `GET`
- **认证**: 需要

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ids | String | 是 | 监测数据ID，多个用逗号分隔 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": null
}
```

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getMonitoringData(params) {
  return request({
    url: '/data_management/monitoring_data/find_data_params',
    method: 'post',
    data: params
  });
}

export function getMonitoringDataById(id) {
  return request({
    url: '/data_management/monitoring_data/find_data_id',
    method: 'post',
    data: { id }
  });
}

export function addMonitoringData(data) {
  return request({
    url: '/data_management/monitoring_data/add_data',
    method: 'post',
    data
  });
}

export function updateMonitoringData(data) {
  return request({
    url: '/data_management/monitoring_data/update_data',
    method: 'post',
    data
  });
}

export function deleteMonitoringData(ids) {
  return request({
    url: `/data_management/monitoring_data/delete_data/${ids}`,
    method: 'get'
  });
}

export function getMonitoringIndicators(areaId) {
  return request({
    url: '/data_management/monitoring_data/indicator_card',
    method: 'get',
    params: { area_id: areaId }
  });
}
```
