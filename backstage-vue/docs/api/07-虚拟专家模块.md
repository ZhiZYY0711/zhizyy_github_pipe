# 虚拟专家模块 API

## 概述

虚拟专家模块提供AI智能问答功能，使用SSE（Server-Sent Events）实现实时流式响应。

---

## 1. SSE连接

### 接口说明

建立SSE连接，用于接收AI实时响应。

### 请求信息

- **URL**: `/manager/virtual-expert/stream`
- **Method**: `GET`
- **认证**: 需要
- **Content-Type**: `text/event-stream`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| conversationId | String | 是 | 会话ID |
| userId | String | 否 | 用户ID |
| token | String | 否 | JWT Token |

### 响应格式

SSE事件流格式：

```
event: system
data: {"conversationId":"session_123","userId":"user_001","content":"SSE连接建立成功","timestamp":1704067200000}

event: status
data: {"status":"processing","message":"AI正在思考..."}

event: delta
data: {"content":"这是","timestamp":1704067200100}

event: delta
data: {"content":"AI","timestamp":1704067200200}

event: done
data: {"content":"完整的AI回复内容","timestamp":1704067200300}

event: error
data: {"error":"错误信息","timestamp":1704067200400}
```

### 事件类型

| 事件类型 | 说明 |
|---------|------|
| system | 系统消息（连接建立、断开等） |
| status | 状态更新（AI思考中、处理中等） |
| delta | 增量内容（流式输出的文本片段） |
| done | 完成消息（完整内容） |
| error | 错误消息 |

## 2. 发送消息

### 接口说明

向AI发送消息，触发AI响应。

### 请求信息

- **URL**: `/manager/virtual-expert/messages`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "conversationId": "session_123",
  "userId": "user_001",
  "type": "chat",
  "content": "请帮我分析一下管道压力异常的原因"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| conversationId | String | 是 | 会话ID |
| userId | String | 否 | 用户ID |
| type | String | 否 | 消息类型（chat） |
| content | String | 是 | 消息内容 |

### 响应参数

```json
{
  "success": true,
  "message": "消息已接收，AI处理中",
  "conversationId": "session_123",
  "timestamp": 1704067200000
}
```

## 前端集成建议

### 1. SSE连接管理

```javascript
class VirtualExpertService {
  constructor() {
    this.eventSource = null;
    this.conversationId = null;
  }
  
  connect(conversationId, userId, token) {
    if (this.eventSource) {
      this.disconnect();
    }
    
    this.conversationId = conversationId;
    const url = `/manager/virtual-expert/stream?conversationId=${encodeURIComponent(conversationId)}&userId=${encodeURIComponent(userId)}&token=${encodeURIComponent(token)}`;
    
    this.eventSource = new EventSource(url);
    
    this.eventSource.addEventListener('system', (event) => {
      const data = JSON.parse(event.data);
      console.log('系统消息:', data);
    });
    
    this.eventSource.addEventListener('delta', (event) => {
      const data = JSON.parse(event.data);
      this.onDelta(data.content);
    });
    
    this.eventSource.addEventListener('done', (event) => {
      const data = JSON.parse(event.data);
      this.onDone(data.content);
    });
    
    this.eventSource.addEventListener('error', (event) => {
      const data = JSON.parse(event.data);
      this.onError(data.error);
    });
    
    this.eventSource.onerror = (error) => {
      console.error('SSE连接错误:', error);
      this.reconnect();
    };
  }
  
  disconnect() {
    if (this.eventSource) {
      this.eventSource.close();
      this.eventSource = null;
    }
  }
  
  async sendMessage(content) {
    const response = await fetch('/manager/virtual-expert/messages', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${localStorage.getItem('token')}`
      },
      body: JSON.stringify({
        conversationId: this.conversationId,
        userId: localStorage.getItem('username'),
        type: 'chat',
        content
      })
    });
    
    return response.json();
  }
  
  onDelta(content) {
    // 子类实现
  }
  
  onDone(content) {
    // 子类实现
  }
  
  onError(error) {
    // 子类实现
  }
  
  reconnect() {
    // 重连逻辑
  }
}
```

### 2. 聊天界面组件

```vue
<template>
  <div class="virtual-expert-page">
    <div class="chat-container">
      <div class="chat-messages" ref="messagesContainer">
        <div
          v-for="message in messages"
          :key="message.id"
          class="message"
          :class="`message--${message.role}`"
        >
          <div class="message__avatar">
            <el-avatar :icon="message.role === 'user' ? 'el-icon-user' : 'el-icon-cpu'" />
          </div>
          <div class="message__content">
            <div class="message__text">{{ message.content }}</div>
            <div class="message__time">{{ message.timestamp }}</div>
          </div>
        </div>
        
        <div v-if="aiTyping" class="message message--assistant">
          <div class="message__avatar">
            <el-avatar icon="el-icon-cpu" />
          </div>
          <div class="message__content">
            <div class="message__text typing">{{ currentResponse }}</div>
          </div>
        </div>
      </div>
      
      <div class="chat-input">
        <el-input
          v-model="inputMessage"
          type="textarea"
          :rows="3"
          placeholder="请输入您的问题..."
          @keydown.enter.prevent="handleSend"
        />
        <button class="btn btn-primary" @click="handleSend" :disabled="!inputMessage.trim()">
          发送
        </button>
      </div>
    </div>
    
    <div class="chat-history">
      <h3>历史会话</h3>
      <div
        v-for="session in sessions"
        :key="session.id"
        class="session-item"
        @click="loadSession(session)"
      >
        <div class="session-item__title">{{ session.title }}</div>
        <div class="session-item__time">{{ session.time }}</div>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      messages: [],
      inputMessage: '',
      currentResponse: '',
      aiTyping: false,
      sessions: []
    };
  },
  methods: {
    handleSend() {
      if (!this.inputMessage.trim()) return;
      
      const userMessage = {
        id: Date.now(),
        role: 'user',
        content: this.inputMessage,
        timestamp: new Date().toLocaleString()
      };
      
      this.messages.push(userMessage);
      this.aiTyping = true;
      this.currentResponse = '';
      
      // 发送消息到AI
      this.sendMessage(this.inputMessage);
      this.inputMessage = '';
    },
    
    onDelta(content) {
      this.currentResponse += content;
      this.scrollToBottom();
    },
    
    onDone(content) {
      this.aiTyping = false;
      this.messages.push({
        id: Date.now(),
        role: 'assistant',
        content: content,
        timestamp: new Date().toLocaleString()
      });
    },
    
    scrollToBottom() {
      this.$nextTick(() => {
        const container = this.$refs.messagesContainer;
        container.scrollTop = container.scrollHeight;
      });
    }
  }
};
</script>
```
