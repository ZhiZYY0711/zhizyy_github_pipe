# 演练管理模块 API

## 概述

演练管理模块提供应急演练信息的增删改查功能。

---

## 1. 查询演练信息（非ID）

### 接口说明

查询演练信息列表，支持多条件筛选。

### 请求信息

- **URL**: `/manoeuvre/find_manoeuvre_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "name": "消防演练",
  "type": 1,
  "status": 1,
  "area_id": 130100,
  "start_time": "2024-01-01 00:00:00",
  "end_time": "2024-12-31 23:59:59"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | String | 否 | 演练名称 |
| type | Integer | 否 | 演练类型 |
| status | Integer | 否 | 演练状态 |
| area_id | Long | 否 | 区域ID |
| start_time | String | 否 | 开始时间 |
| end_time | String | 否 | 结束时间 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": "1",
      "name": "2024年消防应急演练",
      "type": "消防演练",
      "status": "已完成",
      "area_name": "石家庄市",
      "location": "A区管道站",
      "start_time": "2024-01-15 09:00:00",
      "end_time": "2024-01-15 11:00:00",
      "participants": 25,
      "organizer": "安全管理部",
      "description": "年度消防应急演练"
    }
  ]
}
```

| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | String | 演练ID |
| name | String | 演练名称 |
| type | String | 演练类型 |
| status | String | 演练状态 |
| area_name | String | 区域名称 |
| location | String | 演练地点 |
| start_time | String | 开始时间 |
| end_time | String | 结束时间 |
| participants | Integer | 参与人数 |
| organizer | String | 组织部门 |
| description | String | 演练描述 |

export default {
  methods: {
    getStatusType(status) {
      const map = {
        '计划中': 'default',
        '进行中': 'primary',
        '已完成': 'success'
      };
      return map[status] || 'default';
    }
  }
};
</script>

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getManoeuvres(params) {
  return request({
    url: '/manoeuvre/find_manoeuvre_params',
    method: 'post',
    data: params
  });
}

export function getManoeuvreById(id) {
  return request({
    url: '/manoeuvre/find_manoeuvre_id',
    method: 'get',
    params: { id }
  });
}

export function addManoeuvre(data) {
  return request({
    url: '/manoeuvre/add_manoeuvre',
    method: 'post',
    data
  });
}

export function updateManoeuvre(data) {
  return request({
    url: '/manoeuvre/update_manoeuvre',
    method: 'post',
    data
  });
}

export function deleteManoeuvre(id) {
  return request({
    url: '/manoeuvre/remove_manoeuvre',
    method: 'get',
    params: { id }
  });
}
```
