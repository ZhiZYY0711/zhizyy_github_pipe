# components 目录结构

## 组件说明

- **ChinaMap.vue**: 用于展示中国地图的组件，可能用于区域概览或地理位置相关的数据展示。
- **DeviceInfo.vue**: 设备信息管理组件，用于显示和管理系统中的设备详细信息。
- **EChartsMap.vue**: 基于 ECharts 库实现的地图组件，用于在地图上展示数据，例如管道分布、报警位置等。
- **EChartsPie.vue**: 基于 ECharts 库实现的饼图组件，用于展示数据的比例分布，例如报警级别分布。
- **FlowTrendChart.vue**: 流量趋势图组件，用于展示随时间变化的流量数据趋势。
- **InspectionLog.vue**: 巡检日志组件，用于记录和展示巡检人员的巡检活动和结果。
- **LogRecord.vue**: 日志记录组件，用于显示系统操作日志或事件日志。

- **RepairmanInfo.vue**: 检修员信息管理组件，用于显示和管理检修人员的详细信息。
- **SensorLog.vue**: 传感器日志组件，用于记录和展示传感器采集到的数据日志。
- **TaskManagement.vue**: 任务管理组件，用于创建、分配、跟踪和管理系统中的各项任务。
