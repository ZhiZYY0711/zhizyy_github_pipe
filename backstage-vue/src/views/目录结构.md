# Views 目录结构

## 重构后的文件夹结构

### data-visualization/ - 数据可视化模块
- **DataVisualization.vue** - 数据可视化页面，以图表形式展示数据分析结果
  - 使用组件：EChartsMap、EChartsPie（来自 src/components）
- **DataMonitoring.vue** - 数据监控页面，实时监控系统数据和状态
  - 使用组件：AreaDetails、PipelineDetails（views/details 子组件）；未复用 src/components
- **TaskDetails.vue** - 任务详情页面，显示具体任务的详细信息
  - 使用组件：无 src/components 复用；页面内直接使用 ECharts

### data-management/ - 数据管理模块
- **Monitoring.vue** - 监控页面，综合监控系统各项指标
  - 使用组件：无 src/components 复用
- **Equipment.vue** - 设备管理页面，管理和监控设备状态
  - 使用组件：无 src/components 复用
- **Tasks.vue** - 任务管理页面，管理和分配各种任务
  - 使用组件：无 src/components 复用
- **Maintenance.vue** - 维护管理页面，管理设备维护计划和记录
  - 使用组件：无 src/components 复用

### virtual-expert/ - 虚拟专家模块
- **VirtualExpert.vue** - 虚拟专家页面，提供专家系统和智能建议
  - 使用组件：无 src/components 复用

### simulation-drill/ - 模拟演练模块
- **SimulationDrill.vue** - 模拟演练页面，进行应急演练和培训
  - 使用组件：无 src/components 复用

### emergency/ - 应急响应模块
- **AccidentResponse.vue** - 事故响应页面，处理紧急事故的响应和处理流程
  - 使用组件：无 src/components 复用

### logs/ - 日志管理模块
- **LogRecording.vue** - 日志记录页面，记录和查看系统操作日志
  - 使用组件：无 src/components 复用

### details/ - 详情页面模块
- **AreaDetails.vue** - 区域详情页面，显示特定区域的详细信息
  - 使用组件：无 src/components 复用；页面内直接使用 ECharts
- **PipelineDetails.vue** - 管道详情页面，显示管道的详细信息和状态
  - 使用组件：无 src/components 复用；页面内直接使用 ECharts

### 根目录
- **LoginPage.vue** - 登录页面，负责用户登录认证
  - 使用组件：无 src/components 复用
- **Layout.vue** - 布局组件，定义应用的整体布局结构
  - 使用组件：WeatherDate（来自 src/components）
