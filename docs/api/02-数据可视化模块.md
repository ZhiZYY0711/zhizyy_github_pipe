# 数据可视化模块 API

本文档按当前 Spring Controller 与 DTO 记录真实合同。

## 全局 KPI

### GET `/data_visualization/all/whole_kpi`

返回：

```json
{
  "sensor_numbers": 2318,
  "abnormal_sensor_numbers": 43,
  "warnings": 7,
  "underway_task": 26,
  "overtime_task": 3
}
```

### GET `/data_visualization/all/repairman_kpi`

返回：

```json
{
  "repairman_number": 50,
  "today_task_number": 12,
  "task_rate": 96,
  "average_time": 3
}
```

### GET `/data_visualization/all/running_water_alarm`

Query：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| area_id | Long | 否 | 区域 ID |
| start_time | Long | 否 | 开始时间戳，毫秒 |
| end_time | Long | 否 | 结束时间戳，毫秒 |

返回数组字段：`id`、`sensor_id`、`sensor_name`、`area_id`、`area_name`、`flow`、`time`。

## 任务详情图表

统一路径前缀：`/data_visualization/task_details`。请求体通用字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| area_id | Long | 是 | 区域 ID |
| start_time | Long | 否 | 开始时间戳，毫秒 |
| end_time | Long | 否 | 结束时间戳，毫秒 |

### POST `/task_status`

返回数组字段：`status`、`count`。

### POST `/task_nums`

返回数组字段：`time`、`count`。

### POST `/task_contrast`

返回数组字段：`area_name`、`count`。

### POST `/KPI_list`

额外请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| type | Integer | 是 | 0 完成任务数，1 平均响应时间，2 平均完成时间 |

返回数组字段：`data`、`name`、`repairman_id`。

## 监测数据图表

统一路径前缀：`/data_visualization/data_monitoring`。

### POST `/area_data`

请求字段：`area_id`、`start_time`、`end_time`。

返回数组字段：`ave_flow`、`ave_pressure`、`ave_temperature`、`ave_vibration`、`time`、`area_name`。

### POST `/pipe_data`

请求字段：`segment_id` 或 `segment_ids`、`start_time`、`end_time`。

返回数组字段：`ave_flow`、`ave_pressure`、`ave_temperature`、`ave_vibration`、`time`、`pipe_name`。

### GET `/key_indicators`

Query：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | String | 否 | 管线 ID |
| segment_id | String | 否 | 单个管段 ID |
| segment_ids | String | 否 | 连续管段 ID，逗号分隔 |

返回：

```json
{
  "ave_temperature": 42.6,
  "temperature_status": 1,
  "ave_flow": 4239,
  "flow_status": 0,
  "ave_pressure": 1.18,
  "pressure_status": 0,
  "ave_vibration": 3.2,
  "vibration_status": 1
}
```
