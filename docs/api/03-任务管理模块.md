# 任务管理模块 API

路径前缀：`/data_management/tasks`。

## POST `/find_task_params`

请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码 |
| page_size | Integer | 否 | 每页数量 |
| inspection_id | Long | 否 | 巡检 ID |
| repairman_id | Long | 否 | 检修员 ID |
| area_id | Long | 否 | 区域 ID |
| pipe_id | Long | 否 | 管线 ID |
| pipe_segment_id | Long | 否 | 管段 ID |
| type | Long | 否 | 任务类型 |
| priority | Long | 否 | 优先级 |
| status | Long | 否 | 状态 |
| start_time | String | 否 | 发布时间起始，`yyyy-MM-dd HH:mm:ss` |
| end_time | String | 否 | 发布时间结束，`yyyy-MM-dd HH:mm:ss` |

返回分页字段：`records`、`total`、`pageNum`、`pageSize`、`pages`。

任务记录字段：`id`、`title`、`type`、`area_name`、`pipe_name`、`pipe_segment_id`、`pipe_segment_name`、`priority`、`status`、`assignee`、`phone`、`public_time`、`response_time`、`accomplish_time`、`feedback_information`。

## POST `/find_task_id`

请求体：`{ "id": 801 }`。返回单条任务记录。

## POST `/add_task`

请求字段：`inspection_id`、`repairman_id`、`area_id`、`pipe_id`、`pipe_segment_id`、`task_name`、`type`、`priority`、`status`、`feedback_information`。

## POST `/update_task`

请求字段：`id`、`inspection_id`、`repairman_id`、`area_id`、`pipe_id`、`pipe_segment_id`、`task_name`、`type`、`priority`、`status`、`public_time`、`response_time`、`accomplish_time`、`feedback_information`。

## GET `/delete_task/{id}`

按任务 ID 删除。

## GET `/Indicator_card`

Query：`area_id` 可选。

返回：

```json
{
  "total": "150",
  "pending": "15",
  "in_progress": "8",
  "completed": "120",
  "urgent": "7"
}
```
