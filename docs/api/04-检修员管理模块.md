# 检修员管理模块 API

路径前缀：`/data_management/repairman`。

## POST `/find_repairman_params`

请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码 |
| page_size | Integer | 否 | 每页数量 |
| name | String | 否 | 姓名 |
| sex | String | 否 | 性别 |
| phone | String | 否 | 电话 |
| min_age | Integer | 否 | 最小年龄 |
| max_age | Integer | 否 | 最大年龄 |
| area_id | Long | 否 | 区域 ID |
| entry_start_time | String | 否 | 入职开始时间 |
| entry_end_time | String | 否 | 入职结束时间 |

返回：

```json
{
  "total": 50,
  "records": [
    {
      "id": "501",
      "name": "张三",
      "age": "35",
      "sex": "0",
      "phone": "13800000000",
      "area_name": "石家庄市",
      "entry_time": "2024-01-01"
    }
  ]
}
```

## POST `/find_repairman_id`

请求体：`{ "id": "501" }`。返回单个检修员。

## POST `/add_repairman`

请求字段：`name`、`age`、`sex`、`phone`、`area_id`、`entry_time`。

## POST `/update_repairman`

请求字段：`id`、`name`、`age`、`sex`、`phone`、`area_id`、`entry_time`。

## GET `/delete_repairman/{ids}`

`ids` 支持逗号分隔。

## GET `/Indicator_card`

Query：`area_id` 可选。

返回：

```json
{
  "total": "50",
  "male": "30",
  "female": "20",
  "avg_age": "35",
  "new_this_month": "4"
}
```
