# 检修员管理模块 API

## 概述

检修员管理模块提供检修员信息的增删改查功能。

---

## 1. 查询检修员（非ID）

### 接口说明

分页查询检修员列表，支持多条件筛选。

### 请求信息

- **URL**: `/data_management/repairman/find_repairman_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "page": 1,
  "page_size": 10,
  "name": "张",
  "sex": "0",
  "min_age": "25",
  "max_age": "45",
  "area_id": "130100",
  "entry_start_time": "2020-01-01",
  "entry_end_time": "2024-12-31",
  "phone": "138"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | Integer | 是 | 页码，从1开始 |
| page_size | Integer | 是 | 每页数量 |
| name | String | 否 | 检修员姓名（模糊查询） |
| sex | String | 否 | 性别（0男 1女 2未知） |
| min_age | String | 否 | 最小年龄 |
| max_age | String | 否 | 最大年龄 |
| area_id | String | 否 | 所属区域ID |
| entry_start_time | String | 否 | 入职起始时间 |
| entry_end_time | String | 否 | 入职结束时间 |
| phone | String | 否 | 联系电话（模糊查询） |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": "1",
      "name": "张三",
      "age": "35",
      "sex": "0",
      "phone": "13800138000",
      "area_name": "石家庄市",
      "entry_time": "2020-05-15"
    }
  ]
}
```

| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | String | 检修员ID |
| name | String | 姓名 |
| age | String | 年龄 |
| sex | String | 性别（0男 1女 2未知） |
| phone | String | 联系电话 |
| area_name | String | 所属区域名称 |
| entry_time | String | 入职时间 |

## 5. 删除检修员（按IDs）

### 接口说明

批量删除检修员，支持删除多个。

### 请求信息

- **URL**: `/data_management/repairman/delete_repairman/{ids}`
- **Method**: `GET`
- **认证**: 需要

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ids | String | 是 | 检修员ID，多个用逗号分隔（路径参数） |

示例：`/data_management/repairman/delete_repairman/1,2,3`

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": null
}
```

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getRepairmen(params) {
  return request({
    url: '/data_management/repairman/find_repairman_params',
    method: 'post',
    data: params
  });
}

export function getRepairmanById(id) {
  return request({
    url: '/data_management/repairman/find_repairman_id',
    method: 'post',
    data: { id }
  });
}

export function addRepairman(data) {
  return request({
    url: '/data_management/repairman/add_repairman',
    method: 'post',
    data
  });
}

export function updateRepairman(data) {
  return request({
    url: '/data_management/repairman/update_repairman',
    method: 'post',
    data
  });
}

export function deleteRepairmen(ids) {
  return request({
    url: `/data_management/repairman/delete_repairman/${ids}`,
    method: 'get'
  });
}

export function getRepairmanIndicators(areaId) {
  return request({
    url: '/data_management/repairman/Indicator_card',
    method: 'get',
    params: { area_id: areaId }
  });
}
```
