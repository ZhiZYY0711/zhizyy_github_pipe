# 虚拟专家模块 API

当前后端实现的是 Agent 会话接口，路径前缀同时支持：

- `/virtual-expert/agent`
- `/manager/virtual-expert/agent`

前端生产环境默认走 `/manager/virtual-expert/agent`。历史 `/manager/virtual-expert/stream`、`/manager/virtual-expert/messages` 未在当前后端实现，不应作为业务合同使用。

## 会话

### GET `/sessions`

Query：`limit` 可选。返回 Agent 服务透传的会话列表。

### POST `/sessions`

请求体透传给 Agent 服务，用于创建会话。

### DELETE `/sessions/{sessionId}`

删除指定会话。

### GET `/sessions/{sessionId}/timeline`

Query：`beforeCursor`、`limit` 可选。返回会话时间线。

## 消息与运行

### POST `/sessions/{sessionId}/messages`

请求体透传给 Agent 服务，用于追加用户消息或业务上下文。

### POST `/sessions/{sessionId}/runs`

启动会话运行。返回 `AgentRunResponse`。

### GET `/sessions/{sessionId}/runs/stream`

SSE 流式运行当前会话。

### GET `/sessions/{sessionId}/runs/{runId}/stream`

SSE 流式订阅指定运行。

### GET `/sessions/{sessionId}/events`

Query：`afterSeq` 可选。返回会话事件。

### GET `/sessions/{sessionId}/runs/{runId}/events`

Query：`afterSeq` 可选。返回指定运行事件。

### POST `/sessions/{sessionId}/runs/{runId}/cancel`

取消指定运行。

## 导出

### POST `/exports`

创建导出文件。返回：

```json
{
  "exportId": "exp-xxx",
  "fileName": "virtual-expert.xlsx",
  "contentType": "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
  "size": 1024,
  "downloadUrl": "/manager/virtual-expert/agent/exports/exp-xxx/download"
}
```

### GET `/exports/{exportId}`

查询导出文件元信息。

### GET `/exports/{exportId}/download`

下载导出文件。

## 内部工具代理

`/internal/virtual-expert/tools/*` 为 Agent 工具代理接口，供虚拟专家服务调用，不属于管理端页面业务接口。
