# 日志管理模块 API

## 概述

日志管理模块提供系统日志的查询功能。

---

## 1. 查询日志（非ID）

### 接口说明

分页查询日志列表，支持多条件筛选。

### 请求信息

- **URL**: `/log/find_logs_params`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

```json
{
  "page": 1,
  "page_size": 10,
  "user_id": "admin",
  "operation": "登录",
  "start_time": "2024-01-01 00:00:00",
  "end_time": "2024-01-31 23:59:59"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | Integer | 否 | 页码，从1开始 |
| page_size | Integer | 否 | 每页数量 |
| user_id | String | 否 | 用户ID |
| operation | String | 否 | 操作类型 |
| start_time | String | 否 | 开始时间 |
| end_time | String | 否 | 结束时间 |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": "1",
        "user_id": "admin",
        "user_name": "管理员",
        "operation": "登录",
        "method": "POST",
        "url": "/login",
        "ip": "192.168.1.100",
        "location": "河北省石家庄市",
        "status": "成功",
        "error_msg": null,
        "duration": 150,
        "create_time": "2024-01-15 10:30:00"
      }
    ],
    "total": 1000,
    "page": 1,
    "pageSize": 10
  }
}
```

## 2. 查询日志（ID）

### 接口说明

根据ID查询日志详细信息。

### 请求信息

- **URL**: `/log/find_logs_id`
- **Method**: `POST`
- **认证**: 需要

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | Long | 是 | 日志ID（查询参数） |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "id": "1",
    "user_id": "admin",
    "user_name": "管理员",
    "operation": "登录",
    "method": "POST",
    "url": "/login",
    "params": "{\"username\":\"admin\"}",
    "ip": "192.168.1.100",
    "location": "河北省石家庄市",
    "status": "成功",
    "error_msg": null,
    "duration": 150,
    "create_time": "2024-01-15 10:30:00"
  }
}
```

---

## 3. 获取日志指标卡

### 接口说明

获取日志统计指标数据。

### 请求信息

- **URL**: `/log/indicator_card`
- **Method**: `GET`
- **认证**: 需要

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| area_id | Long | 否 | 区域ID |

### 响应参数

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "total": 10000,
    "today": 150,
    "success": 9800,
    "failed": 200,
    "avg_duration": 125
  }
}
```

### 2. API 封装

```javascript
import request from '@/utils/request';

export function getLogs(params) {
  return request({
    url: '/log/find_logs_params',
    method: 'post',
    data: params
  });
}

export function getLogById(id) {
  return request({
    url: '/log/find_logs_id',
    method: 'post',
    params: { id }
  });
}

export function getLogIndicators(areaId) {
  return request({
    url: '/log/indicator_card',
    method: 'get',
    params: { area_id: areaId }
  });
}
```
