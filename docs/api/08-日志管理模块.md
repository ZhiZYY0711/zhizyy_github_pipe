# 日志管理模块 API

路径前缀：`/log`。

## POST `/find_logs_params`

请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| area_id | Long | 否 | 区域 ID |
| operation_time_min | Long | 否 | 操作时间下限，毫秒时间戳 |
| operation_time_max | Long | 否 | 操作时间上限，毫秒时间戳 |
| status | Long | 否 | 操作状态 |
| type | Long | 否 | 操作员类型，0 管理员，1 检修员 |
| page | Integer | 否 | 页码 |
| page_size | Integer | 否 | 每页数量 |

返回分页字段：`records`、`total`、`pageNum`、`pageSize`、`pages`。

日志记录字段：`id`、`user_id`、`username`、`type`、`user_type_text`、`operate`、`ip_address`、`status`、`status_text`、`period`、`operate_time`、`details`。

## POST `/find_logs_id`

Query：`id` 必填。返回单条日志记录。

## GET `/indicator_card`

Query：`area_id` 可选。

返回：

```json
{
  "debugging": 10,
  "error": 2,
  "all": 100,
  "total": 100,
  "success": 98,
  "failed": 2,
  "avg_duration": 125,
  "today": 8,
  "warning": 3
}
```
