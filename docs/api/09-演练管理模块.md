# 演练管理模块 API

路径前缀：`/manoeuvre`。

## POST `/find_manoeuvre_params`

请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| area_id | Long | 否 | 区域 ID |
| type | Long | 否 | 0 状态模拟，1 事故模拟，2 紧急事件，3 日常演练 |
| status | Long | 否 | 0 成功，1 失败，2 进行中 |
| start_time_min | Long | 否 | 开始时间下限，毫秒时间戳 |
| start_time_max | Long | 否 | 开始时间上限，毫秒时间戳 |
| end_time_min | Long | 否 | 结束时间下限，毫秒时间戳 |
| end_time_max | Long | 否 | 结束时间上限，毫秒时间戳 |
| page | Integer | 否 | 页码 |
| page_size | Integer | 否 | 每页数量 |

返回数组字段为 camelCase：`id`、`name`、`areaId`、`areaName`、`location`、`startTime`、`endTime`、`organizer`、`status`、`type`、`details`、`repairmans`。

## GET `/find_manoeuvre_id`

Query：`id` 必填。返回演练数组。

## POST `/add_manoeuvre`

请求字段：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| area_id | Long | 是 | 区域 ID |
| detail | String | 否 | 详情 |
| repairmans | Long[] | 是 | 参与检修员 ID |
| start_time | Long | 是 | 开始时间，毫秒时间戳 |
| status | Long | 是 | 状态 |
| type | Long | 是 | 类型 |

## POST `/update_manoeuvre`

请求字段：`id`、`areaId`、`detail`、`endTime`、`repairmans`、`startTime`、`status`、`type`。

> 更新接口当前使用 camelCase 的 `areaId`、`startTime`、`endTime`。

## GET `/remove_manoeuvre`

Query：`id` 必填。
