# 页面/模拟演练接口

## POST 发布演练

POST /domain/manager/manoeuvre/add_manoeuvre

插入演练信息 点击模拟开始按钮时触发

> Body 请求参数

```json
{
  "area_id": 210681,
  "start_time": 1749687991687,
  "status": 0,
  "type": 1,
  "detail": "commodo magna eu velit ipsum",
  "repairmans": [
    52,
    24,
    27
  ]
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer| 是 | 区域id|none|
|» start_time|body|integer| 是 | 开始时间|时间戳|
|» status|body|integer| 是 | 状态|none|
|» type|body|integer| 是 | 类型|none|
|» detail|body|string| 否 | 详情|none|
|» repairmans|body|[integer]| 是 | 参加演习的检修员id|none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {}
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|object|true|none||响应数据|

## POST 查询演练信息（非id）

POST /domain/manager/manoeuvre/find_manoeuvre_params

通过条件查询的参数 获取演习信息
进入页面时触发一次

> Body 请求参数

```json
{
  "area_id": 0,
  "start_time_min": 0,
  "start_time_max": 0,
  "end_time_min": 0,
  "end_time_max": 0,
  "status": 0,
  "type": 0,
  "page": 0,
  "page_size": 0
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer| 是 | 任务区域|none|
|» start_time_min|body|integer| 否 | 最早开始时间|时间戳|
|» start_time_max|body|integer| 否 | 最晚开始时间|时间戳|
|» end_time_min|body|integer| 否 | 最早结束时间|时间戳|
|» end_time_max|body|integer| 否 | 最晚结束时间|时间戳|
|» status|body|integer| 否 | 演练状态|0 成功 1 失败 2 进行中|
|» type|body|integer| 否 | 类型|0 状态模拟 1 事故模拟 2 紧急事故 3 日常作训|
|» page|body|integer| 是 | 页码|none|
|» page_size|body|integer| 是 | 每页数据量|none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "id": 0,
      "area_id": 0,
      "area_name": "string",
      "start_time": 0,
      "end_time": 0,
      "status": 0,
      "type": 0,
      "details": "string",
      "repairmans": [
        {
          "repairman_id": 0,
          "repairman_name": "string"
        }
      ]
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|false|none||none|
|»» id|integer|true|none||演练id|
|»» area_id|integer|true|none|区域id|none|
|»» area_name|string|true|none|区域名称|none|
|»» start_time|integer|true|none|开始时间|时间戳|
|»» end_time|integer|false|none|结束时间|时间戳|
|»» status|integer|true|none|状态|0 成功 1 失败 2 进行中|
|»» type|integer|true|none|类型|0 状态模拟 1 事故模拟 2 紧急事故 3 日常作训|
|»» details|string|false|none|描述|none|
|»» repairmans|[object]|true|none|参加演习的检修员|none|
