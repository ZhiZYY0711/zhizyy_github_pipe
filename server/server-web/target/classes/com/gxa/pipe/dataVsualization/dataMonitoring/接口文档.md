## POST 区域四维度数据

POST /domain/manager/data_visualization/data_monitoring/area_details/area_data

> Body 请求参数

```json
{
  "area_id": 0,
  "start_time": "string",
  "end_time": "string"
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer¦null| 否 | 区域id|为空时默认查所有区域的总和|
|» start_time|body|string¦null| 否 | 开始时间|为空时不限制开始时间|
|» end_time|body|string¦null| 否 | 结束时间|为空时不限制结束时间|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "ave_flow": "string",
      "ave_pressure": "string",
      "ave_temperature": "string",
      "ave_vibration": "string",
      "time": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||响应数据|
|»» ave_flow|string|true|none||流量|
|»» ave_pressure|string|true|none||压力|
|»» ave_temperature|string|true|none||温度|
|»» ave_vibration|string|true|none||震动|
|»» time|string|true|none||日期|
