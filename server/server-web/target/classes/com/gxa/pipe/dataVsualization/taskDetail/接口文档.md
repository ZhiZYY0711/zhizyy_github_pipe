## POST 任务状态分布

POST /data_visualization/task_details/task_status

获取任务状态分布 进入页面触发

> Body 请求参数

```json
{
  "area_id": 230108
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer| 是 | 区域id|none|
|» start_time|body|integer| 否 | 开始时间|为空时不限制开始时间 时间戳|
|» end_time|body|integer| 否 | 结束时间|为空时不限制结束时间 时间戳|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "status": 0,
      "count": 0
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||响应数据|
|»» status|integer|true|none|状态|none|
|»» count|integer|true|none|数量统计|none|

## POST KPI榜单

POST /data_visualization/task_details/KPI_list

获取KPI榜单数据

> Body 请求参数

```json
{
  "area_id": 0,
  "start_time": 0,
  "end_time": 0,
  "type": 0
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer| 是 | 区域id|none|
|» start_time|body|integer| 否 | 开始时间|为空时不限制开始时间 时间戳|
|» end_time|body|integer| 否 | 结束时间|为空时不限制结束时间 时间戳|
|» type|body|integer| 是 | 榜单类型|0 完成任务数 1 平均响应时间 2 平均完成时间|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "repairman_id": 0,
      "name": "string",
      "data": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||响应数据|
|»» repairman_id|integer|true|none||检修员的唯一标识|
|»» name|string|true|none||检修员姓名|
|»» data|string|true|none||具体数据|
