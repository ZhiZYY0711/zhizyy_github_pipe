## GET 全局KPI

GET /domain/manager/data_visualization/all/whole_kpi

获取全局KPI指标卡的相关数据  进入页面时触发

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|area_id|query|string| 否 |区域id|
|Authorization|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "sensor_numbers": 0,
    "abnormal_sensor_numbers": 0,
    "warnings": 0,
    "underway_task": 0,
    "overtime_task": 0
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|object|true|none||响应数据|
|»» sensor_numbers|integer|false|none||传感器总数|
|»» abnormal_sensor_numbers|integer|false|none||异常/离线的传感器|
|»» warnings|integer|false|none||今日高危/危险报警次数|
|»» underway_task|integer|false|none||进行中的任务数|
|»» overtime_task|integer|false|none||超时/未处理的任务数|

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|area_id|query|string| 否 |区域id|
|Authorization|header|string| 否 |none|

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [

      "id": "string",
```
