# 页面/日志管理接口

## GET 指标卡

GET /domain/manager/log/Indicator_card

获取日志记录的指标卡

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|area_id|query|string| 否 |区域id|
|Authorization|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "all": 0,
  "warning": 0,
  "error": 0,
  "debugging": 0,
  "today": 0
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» all|integer|true|none||全部日志|
|» warning|integer|true|none||警告日志|
|» error|integer|true|none||错误日志|
|» debugging|integer|true|none||调试日志|
|» today|integer|true|none||今日日志|

## POST 查询日志信息（非id）

POST /domain/manager/log/find_logs_params

通过参数查询日志记录

> Body 请求参数

```json
{
  "area_id": 81,
  "type": 0,
  "page": 1,
  "page_size": 20
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|string| 否 ||none|
|body|body|object| 是 ||none|
|» area_id|body|integer| 是 | 区域id|none|
|» type|body|integer| 否 | 操作员类型|none|
|» status|body|integer| 否 | 操作的状态|none|
|» operation_time_min|body|integer| 否 | 操作时间最小值|为空时不限制最小时间 时间戳|
|» operation_time_max|body|integer| 否 | 操作时间最小值|为空时不限制最大时间 时间戳|
|» page|body|integer| 是 | 页码|none|
|» page_size|body|integer| 是 | 每页数据量|none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "id": 0,
      "user_id": 0,
      "type": 0,
      "status": 0,
      "username": "string",
      "ip_address": "string",
      "operate": "string",
      "operate_time": 0,
      "period": 0,
      "details": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||响应数据|
|»» id|integer|true|none|日志的唯一标识|none|
|»» user_id|integer|true|none|管理员或检修员的id|none|
|»» type|integer|true|none|类型|0 管理员 1 检修员|
|»» status|integer|true|none|操作的状态|none|
|»» username|string|true|none|管理员或检修员的用户名|none|
|»» ip_address|string|true|none|操作者的ip地址|none|
|»» operate|string|true|none|进行的操作|none|
|»» operate_time|integer|true|none|操作时间|none|
|»» period|integer|true|none|持续时间|none|
|»» details|string|false|none|操作详情|none|
