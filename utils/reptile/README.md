# 地理边界数据爬虫

## 功能说明

这个爬虫用于从阿里云DataV获取中国行政区划的地理边界数据，支持省、市、区三级行政区划。

## 文件说明

- `geo_crawler.py` - 主爬虫脚本
- `code.txt` - 区划代码映射文件
- `geo_data/` - 下载的地理数据存储目录
  - `province/` - 省级数据
  - `city/` - 市级数据  
  - `district/` - 区级数据
- `crawler.log` - 爬虫运行日志

## 使用方法

### 1. 基本使用

```bash
python geo_crawler.py
```

### 2. 自定义参数

可以在 `main()` 函数中修改以下参数：

```python
code_file = "code.txt"          # 区划代码文件路径
output_dir = "geo_data"         # 输出目录
request_delay = 1.0             # 请求间隔（秒）
```

## URL规则

根据区划级别自动构建不同的URL：

- **省级/市级**: `https://geo.datav.aliyun.com/areas_v3/bound/{code}_full.json`
  - 例如: `110000_full.json` (北京市)
  
- **区级**: `https://geo.datav.aliyun.com/areas_v3/bound/{code}.json`
  - 例如: `110101.json` (东城区)

## 区划级别判断

- 省级：代码后4位为0000 (如110000)
- 市级：代码后2位为00 (如130100)  
- 区级：6位数字 (如110101)

## 功能特性

- ✅ 自动识别区划级别并构建正确的URL
- ✅ 按级别分类保存文件
- ✅ 支持断点续传（跳过已存在的文件）
- ✅ 详细的日志记录
- ✅ 错误重试机制
- ✅ 请求间隔控制
- ✅ 统计信息输出

## 输出格式

下载的文件为标准GeoJSON格式，包含：
- 地理边界坐标
- 行政区划信息
- 中心点坐标
- 层级关系等

## 注意事项

1. 请合理设置请求间隔，避免对服务器造成过大压力
2. 全量下载约3200+个区划，建议在网络稳定时运行
3. 如果中途中断，重新运行会自动跳过已下载的文件
4. 建议定期备份下载的数据

## 错误处理

- 网络错误：自动重试最多3次
- JSON解析错误：记录日志并跳过
- 文件保存错误：记录日志并跳过
- 支持Ctrl+C中断并显示统计信息

## 日志说明

爬虫会生成详细的日志文件 `crawler.log`，包含：
- 下载进度
- 成功/失败记录
- 错误详情
- 统计信息
