# 油气管道监测管理系统项目结构梳理

## Views目录结构

Views目录包含了应用的主要页面组件：

- AreaDetails.vue - 区域详情页面
- Dashboard.vue - 仪表板页面
- DataVisualization.vue - 数据可视化页面
- MainPage.vue - 主页面
- PipelineDetails.vue - 管道详情页面
- TaskDetails.vue - 任务详情页面

## Components目录结构

Components目录包含了可复用的UI组件：

- ChinaMap.vue - 中国地图组件
- DeviceInfo.vue - 设备信息组件
- FlowTrendChart.vue - 流量趋势图表组件
- InspectionLog.vue - 巡检日志组件
- LogRecord.vue - 日志记录组件
- LoginPage.vue - 登录页面组件
- MainLayout.vue - 主布局组件
- MonitoringData.vue - 监测数据组件
- RepairmanInfo.vue - 维修人员信息组件
- SensorLog.vue - 传感器日志组件
- TaskManagement.vue - 任务管理组件

以上是油气管道监测管理系统的前端项目结构梳理。
