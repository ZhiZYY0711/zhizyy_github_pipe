## POST 管理员登录

POST /manager/login

进行管理员登录  点击登录按钮时触发

> Body 请求参数

```json
{
  "username": "admin729",
  "password": "0192023a7bbd73250516f069df18b500",
  "code": "46h2s"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|
|» username|body|string| 是 |用户名|
|» password|body|string| 是 |密码 前端进行加密后传输|
|» code|body|string| 是 |验证码 6位数|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "username": "string",
    "jwt": "string"
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||状态码|
|» message|string|true|none||相应描述|
|» data|object|true|none||响应数据|
|»» username|string|true|none||用户名|
|»» jwt|string|true|none||jwt令牌|
