## POST 查询检修员（非id）

POST /data_management/repairman/find_repairman_params

查询检修员 支持条件查询 通过非检修员id的参数
进入页面时触发一次 点击搜索按钮时触发一次

> Body 请求参数

```json
{
  "name": "string",
  "sex": "string",
  "min_age": "string",
  "max_age": "string",
  "area_id": "string",
  "entry_start_time": "string",
  "entry_end_time": "string",
  "phone": "string",
  "page": "string",
  "page_size": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 否 |none|
|» name|body|string| 否 |检修员姓名|
|» sex|body|string| 否 |性别（0 男 1 女 2 未知）|
|» min_age|body|string| 否 |最小年龄|
|» max_age|body|string| 否 |最大年龄|
|» area_id|body|string| 否 |所属区域id|
|» entry_start_time|body|string| 否 |入职起始时间|
|» entry_end_time|body|string| 否 |入职结束时间|
|» phone|body|string| 否 |联系电话|
|» page|body|string| 是 |页码，从1开始|
|» page_size|body|string| 是 |每页数量|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "id": "string",
      "name": "string",
      "age": "string",
      "sex": "string",
      "phone": "string",
      "area_name": "string",
      "entry_time": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|成功|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||none|
|»» id|string|true|none||检修员的唯一标识|
|»» name|string|true|none||检修员姓名|
|»» age|string|true|none||年龄|
|»» sex|string|true|none||性别（0 男 1 女）|
|»» phone|string|true|none||联系方式|
|»» area_name|string|true|none||所属区域名称|
|»» entry_time|string|true|none||入职时间|
