## POST 查询任务（非id）

POST /domain/manager/data_management/tasks/find_task_params

查询任务 支持条件查询 通过非任务id的参数
进入页面时触发一次 点击搜索按钮时触发一次

> Body 请求参数

```json
{
  "type": "string",
  "status": "string",
  "priority": "string",
  "area_id": "string",
  "pipe_id": "string",
  "assignee": "string",
  "public_time": "string",
  "page": "string",
  "page_size": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 否 |none|
|» type|body|string| 否 |任务类型|
|» status|body|string| 否 |任务状态（待处理/进行中/已完成/已取消）|
|» priority|body|string| 否 |优先级（紧急/高/中/低）|
|» area_id|body|string| 否 |任务区域|
|» pipe_id|body|string| 否 |任务所属管道|
|» assignee|body|string| 否 |指派检修员姓名|
|» public_time|body|string| 否 |任务发布时间|
|» page|body|string| 是 |页码，从1开始|
|» page_size|body|string| 是 |每页数量|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "id": "string",
      "title": "string",
      "type": "string",
      "area_name": "string",
      "pipe_name": "string",
      "priority": "string",
      "status": "string",
      "assignee": "string",
      "phone": "string",
      "public_time": "string",
      "response_time": "string",
      "accomplish_time": "string",
      "feedback_information": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|成功|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||none|
|»» id|string|true|none||任务唯一标识|
|»» title|string|true|none||任务标题|
|»» type|string|true|none||任务类型|
|»» area_name|string|true|none||任务区域|
|»» pipe_name|string|true|none||任务所属管道|
|»» priority|string|true|none||优先级（紧急/高/中/低）|
|»» status|string|true|none||任务状态（待处理/进行中/已完成/已取消）|
|»» assignee|string|true|none||指派检修员姓名|
|»» phone|string|true|none||联系方式|
|»» public_time|string|true|none||发布时间|
|»» response_time|string|true|none||响应时间|
|»» accomplish_time|string|true|none||任务完成时间|
|»» feedback_information|string|true|none||检修员反馈信息|
