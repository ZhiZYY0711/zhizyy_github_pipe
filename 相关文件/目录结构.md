# views 目录结构

## 页面说明

- **AccidentResponse.vue**: 事故响应页面，用于处理和记录系统中的事故。
- **AreaDetails.vue**: 区域详情页面，展示特定区域的详细信息和相关数据。
- **DataMonitoring.vue**: 数据监控页面，实时监控各种数据指标。
- **DataVisualization.vue**: 数据可视化页面，通过图表、地图等形式展示数据。
- **Equipment.vue**: 设备管理页面，用于管理和查看设备信息。
- **Layout.vue**: 布局组件，作为整个应用的主要布局框架，包含导航和内容区域。
- **LogRecording.vue**: 日志记录页面，显示系统操作日志和事件。
- **Maintenance.vue**: 检修管理页面，用于管理检修任务和检修员信息。
- **Monitoring.vue**: 监控页面，可能包含更详细的监控数据或功能。
- **PipelineDetails.vue**: 管道详情页面，展示特定管道的详细信息和状态。
- **SimulationDrill.vue**: 模拟与演练页面，用于进行系统模拟和演练操作。
- **TaskDetails.vue**: 任务详情页面，展示特定任务的详细信息。
- **Tasks.vue**: 任务列表页面，用于查看和管理所有任务。
- **VirtualExpert.vue**: 虚拟专家页面，可能提供智能分析或辅助决策功能。

# components 目录结构

## 组件说明

- **ChinaMap.vue**: 用于展示中国地图的组件，可能用于区域概览或地理位置相关的数据展示。
- **DeviceInfo.vue**: 设备信息管理组件，用于显示和管理系统中的设备详细信息。
- **EChartsMap.vue**: 基于 ECharts 库实现的地图组件，用于在地图上展示数据，例如管道分布、报警位置等。
- **EChartsPie.vue**: 基于 ECharts 库实现的饼图组件，用于展示数据的比例分布，例如报警级别分布。
- **FlowTrendChart.vue**: 流量趋势图组件，用于展示随时间变化的流量数据趋势。
- **InspectionLog.vue**: 巡检日志组件，用于记录和展示巡检人员的巡检活动和结果。
- **LogRecord.vue**: 日志记录组件，用于显示系统操作日志或事件日志。
- **LoginPage.vue**: 登录页面组件，负责用户登录认证。
- **RepairmanInfo.vue**: 检修员信息管理组件，用于显示和管理检修人员的详细信息。
- **SensorLog.vue**: 传感器日志组件，用于记录和展示传感器采集到的数据日志。
- **TaskManagement.vue**: 任务管理组件，用于创建、分配、跟踪和管理系统中的各项任务。
