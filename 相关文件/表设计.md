## registration  登陆表

| 字段名      | 类型          | 限制                    | 描述                                        |
| ----------- | ------------- | ----------------------- | ------------------------------------------- |
| id          | bigint        | 主键 auto_increment     | 检修员或管理员的唯一标识                    |
| type        | tinyint       | not null                | 用户类型 0 全局管理员 1 区域管理员 2 检修员 |
| username    | varchar（20） | unique not null         | 用户名                                      |
| password    | varchar(255)  | not null                | 密码哈希值（增强安全性）                    |
| create_time | datetime      | not null default now()  | 创建时间                                    |
| update_time | datetime      | not null default now()  | 更新时间                                    |
