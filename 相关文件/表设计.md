## registration  登陆表

| 字段名   | 类型          | 限制            | 描述                                        |
| -------- | ------------- | --------------- | ------------------------------------------- |
| id       | varchar（20） | 主键            | 检修员或管理员的唯一标识                    |
| type     | tinyint       | not null        | 用户类型 0 全局管理员 1 区域管理员 2 检修员 |
| username | varchar（20） | unique not null | 用户名                                      |
| password | varchar(255)  | not null        | 密码哈希值（增强安全性）                    |

---

## inspection  监测数据表

| 字段名           | 类型          | 限制     | 描述                                       |
| ---------------- | ------------- | -------- | ------------------------------------------ |
| id               | varchar（20） | 主键     | 监测数据的唯一标识                         |
| pressure         | decimal       | not null | 压力值                                     |
| temperature      | decimal       | not null | 温度                                       |
| traffic          | decimal       | not null | 流量                                       |
| shake            | decimal       | not null | 震动值                                     |
| data_status      | tinyint       | not null | 当前数据的状态 0 安全 1 良好 2 危险 3 高危 |
| realtime_picture | varchar(300)  |          | 实时图片                                   |
| sensor_id        | varchar（30） | 外键     | 传感器的id                                 |

## sensor  传感器表

| 字段名      | 类型          | 限制     | 描述                                  |
| ----------- | ------------- | -------- | ------------------------------------- |
| id          | varchar（20） | 主键     | 传感器的唯一标识                      |
| area_id     | varchar（20） | 外键     | 传感器所属的区域的id                  |
| pipeline_id | varchar(20)   | 外键     | 传感器所属的管道的id                  |
| status      | tinyint       | not null | 当前传感器的状态 0 正常 1 异常 2 离线 |
| position    | varchar（30） | not null | 传感器的经纬度位置                    |

## repairman  检修员表

| 字段名     | 类型          | 限制     | 描述             |
| ---------- | ------------- | -------- | ---------------- |
| id         | varchar（20） | 主键     | 检修员的唯一标识 |
| name       | varchar（5）  | not null | 姓名             |
| age        | tinyint       | not null | 年龄             |
| sex        | tinyint       | not null | 性别             |
| phone      | char（11）    | not null | 联系方式         |
| entry_time | datetime      | not null | 入职时间         |
| area_id    | varchar（id） | 外键     | 所属区域         |

## task  任务表  日志

| 字段名               | 类型          | 限制     | 描述                                      |
| -------------------- | ------------- | -------- | ----------------------------------------- |
| id                   | varchar（20） | 主键     | 任务的唯一标识                            |
| inspection_id        | varchar（20） | 外键     | 出现危险的监测记录的id                    |
| repairman_id         | varchar（20） | 外键     | 被分配任务的检修员的id                    |
| result               | tinyint       | not null | 任务结果 0 已发布 1 已接取 2已完成 3 异常 |
| public_time          | datetime      | not null | 任务发布时间                              |
| response_time        | datetime      |          | 任务响应时间                              |
| accomplish_time      | datetime      |          | 任务完成时间                              |
| feedback_information | text          |          | 检修员的反馈信息（扩展长度）              |

## journal_sensor  传感器日志表

| 字段名        | 类型          | 限制     | 描述                                  |
| ------------- | ------------- | -------- | ------------------------------------- |
| id            | varchar（20） | 主键     | 传感器的日志的唯一标识                |
| sensor_id     | varchar（20） | 外键     | 传感器的id                            |
| status        | tinyint       | not null | 当前传感器的状态 0 正常 1 异常 2 离线 |

| repairman_id  | varchar（id） | 外键     | 上次检修的检修员的id                  |
