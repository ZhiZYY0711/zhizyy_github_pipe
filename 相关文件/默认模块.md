## POST 查询设备信息（非id）

POST /domain/manager/data_management/equipment/find_equipment_params

查询设备信息 支持条件查询 通过非设备id的参数
进入页面时触发一次 点击搜索按钮时触发一次

> Body 请求参数

```json
{
  "id": "string",
  "type": "string",
  "status": "string",
  "area_id": "string",
  "pipe_id": "string",
  "responsible": "string",
  "last_check_start": "string",
  "last_check_end": "string",
  "page": "string",
  "page_size": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|
|» id|body|string| 否 |设备编号|
|» type|body|string| 否 |设备类型|
|» status|body|string| 否 |设备状态（正常/故障/维护中/离线）|
|» area_id|body|string| 否 |设备所属区域|
|» pipe_id|body|string| 否 |设备所属管道|
|» responsible|body|string| 否 |负责人|
|» last_check_start|body|string| 否 |最后检查起始时间|
|» last_check_end|body|string| 否 |最后检查结束时间|
|» page|body|string| 是 |页码|
|» page_size|body|string| 是 |每页数据量|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": [
    {
      "id": "string",
      "type": "string",
      "area_name": "string",
      "pipe_name": "string",
      "status": "string",
      "last_check": "string",
      "responsible": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应代码|
|» message|string|true|none||响应消息|
|» data|[object]|true|none||响应数据|
|»» id|string|true|none||设备的唯一标识|
|»» type|string|true|none||设备类型|
|»» area_name|string|true|none||设备所属区域名称|
|»» pipe_name|string|true|none||设备所属管道名称|
|»» status|string|true|none||设备状态|
|»» last_check|string|true|none||最后检查时间|
|»» responsible|string|true|none||负责人|
